```
10月17日
今日は、プログラム基礎Ⅱの講義でクイックソートについて学んだ。
クイックソートはピボットが中央値に近いほど配列が半分に分割され計算量が少なくなるようだ。
抜粋した一部の要素らの中央値をピボットに選ぶことで中央値を取りやすくしたり、
分割した配列のうち要素数の少ないほうから処理することでスタックの数を少なくするなどの工夫があるようだ。
基本的にあらかじめ用意された関数を呼び出すのみだが、自分で書いて理解を深めたい。

```
